Deploy AI Tools Daily to GitHub Pages

Follow these steps to deploy the site publicly:

  1. Go to github.com/new and create a new repository named ai-tools-daily (public).
  2. Clone the repository locally:
    3. git clone https://github.com/YOUR_USERNAME/ai-tools-daily.git
cd ai-tools-daily
  3. Copy the files from this site into the repository:
    4. cp -r /root/.openclaw/workspace/ai-news-site/* .
  4. Commit and push:
    5. git add .
git commit -m "Initial AI Tools Daily website"
git push
  5. Go to the repository Settings → Pages → Source: select main branch, root folder.
  6. Wait a minute, then visit https://YOUR_USERNAME.github.io/ai-tools-daily

For automatic updates, add a GitHub Actions workflow: 

# .github/workflows/update.yml
name: Update News
on:
  schedule:
    - cron: '0 8 * * *'  # 8:00 daily
jobs:
  update:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Run update script
        run: python3 update.py
      - name: Commit and push
        run: |
          git add data/news.json
          git commit -m "Auto-update news"
          git push

Once deployed, your site will be available at https://YOUR_USERNAME.github.io/ai-tools-daily

You can also deploy to Vercel/Netlify by uploading the folder directly.
